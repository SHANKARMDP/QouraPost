# 📝 Quora Post Clone (CRUD App)

A simple full-stack web application where users can create, read, update, and delete posts — similar to a mini Quora/Twitter app.

---

## 🚀 Live Demo
👉](https://qourapost-1.onrender.com/posts)

---

## 📌 Features
- ✍️ Create new posts  
- 📖 View all posts  
- 🔍 View post details  
- ✏️ Edit existing posts  
- ❌ Delete posts  
- 🎨 Clean and responsive UI  

---

## 🛠️ Tech Stack
- Node.js  
- Express.js  
- EJS (Template Engine)  
- HTML5  
- CSS3  
- UUID (for unique IDs)  
- Method-Override

## 🌐 Deployment
This project is deployed on Render.

To deploy:
1. Push code to GitHub  
2. Connect repo to Render  
3. Set:
   - Build Command: `npm install`
   - Start Command: `node index.js`  
4. Deploy 🚀
